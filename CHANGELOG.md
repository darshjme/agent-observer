# Changelog

All notable changes to `agent-observer` are documented here.

## [1.0.0] — 2026-03-25

### Added
- `Observable` — value container with subscribe/unsubscribe/notify
- `ObservableDict` — dict with set/delete change events
- `ReactiveProperty` — descriptor for reactive class attributes
- `ObserverGroup` — fluent multi-observable subscription manager
- 36 pytest tests (100% passing)
- Full README with reactive agent monitoring example
